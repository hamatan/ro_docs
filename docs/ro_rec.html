#jRO を録画してSNSで公開する 1/2

基本的な流れは アマレコ か OBS でゲーム画面を録画し、aviutlでSNS用にエンコードする
画面解像度 1920x1080以上、4コア以上＆3GHz以上のCPUを搭載したデスクトップ型PCが望ましい


録画方法

OBSにジャンプ (推奨)
CPU負荷：無～高、HDD負荷：低、それなりに劣化、4分で300MB位
ハードウェアエンコードが使える場合も有る（ノートならこれで）

アマレコにジャンプ
CPU負荷：中、HDD負荷：高、無劣化、4分で10GB位

動画編集ソフトで公開できる長さ、サイズに編集して
エンコード(変換)する → aviutlにジャンプ


OBS編

  1. 録画ソフトのインストール
    要るもの：OBS 1つ
    2019/01 現在は OBS-Studio-22.0.2-Full-Installer-x64.exe を普通にインストールすればおｋ

  2. 録画ソフトの設定
    初回起動時は自動構成ウィザードを使う方が無難


    録画の為に最適化し配信はしない を選択


    映像設定は キャンバス：現在の値、FPS：可能なら60を優先 次へ
    性能測定が始まるのでしばし待つ、オワたら設定を適用したらおｋ


    OBSのメイン画面に戻ってきたらファイル→設定で細かい設定をする
    出力の欄を開いたら、出力モードを詳細 にして録画のタブを開く
    録画ファイルのパスをCドライブ以外にするのが望ましい (USB3.0 HDD等)

    エンコーダの所を GeForce にやらせるなら NVENC H.264
    または Core i3/i5/i7 にやらせるなら QuickSyncVideo H.264 (ノートならこれで)

    ターゲット quality
    プロファイル baseline
    レート制御 VBR
    ビットレート 6000 / 最大ビットレート10000 くらい



    エンコーダの項目に QuickSync や NVENC が見当たらない場合や、ソフトウェアエンコードする場合は
    出力モードを基本、録画品質を超高品質、録画フォーマットをmp4にしておく

    出力モードを詳細設定する場合(クリックで展開)

    出力のタブで出力モードを詳細にして、録画のタブを開く

    録画ファイルのパスをCドライブ以外が望ましい(USB3.0 HDD等)
    録画フォーマット mp4、エンコーダ x264
    ビットレート 6000～10000 (録る解像度による)
    プリセット fast～slower (PCのスペックによる fastのが軽い)
    プロファイル high
    x264オプションに bframes=0 を入力しておく


    音声のタブでサンプリングレート 48KHz、デスクトップ音声デバイス以外を Disable(無効) に


    映像のタブで基本解像度と出力解像度をROの解像度と同じサイズにする
    1024x768 とか書けばおｋ　xは小文字のエックスな


    詳細設定のタブでカラーフォーマット NV12、YUV色空間 601、色範囲 全部 にする
    きっと自動で減色してくれるはず・・・　自動で音量を下げる機能を無効 にチェックを入れる


  3. 取り込み
    OBSのメインウィンドウで左下のソースにある＋をクリックして
    ウィンドウキャプチャを選択する


    なんか適当な名前を付ける


    ウィンドウ から Ragexe.exe を選択する
    カーソルをキャプチャ のチェックを入れる


  4. 録画 (次回起動以降はここから)
    OBS本体を横なり下なり適当にはみ出して置いて、タスクトレイの巴紋みたいなアイコンを
    右クリックして 録画/停止 を選べばおｋ


    画質は ソフトエンコ > GeForce(NVENC) > Intel Core i3/i5/i7(QuickSync)
    NVENCとQuickSync は HWエンコードなので、動画を録る負荷がほぼ無いのでノート向け


  5. その他
    /bgm でBGM切っといた方が後で楽
    RO内でrecも同時にやっとくと撮り直しも楽だが、2回見るのめんどいから直で録っちゃってもおｋ
    直で録る場合、カーソルは自キャラやPCを避けて動かすと後で楽 (PT欄からスキルを使用したり)

    基本的にPCで鳴る音、エラー音とかブラウザの音とかdiscordの通知とかの音も入るので
    閉じるなり取り込み中にしてどうぞ

    エンコード編にジャンプ


アマレコ編

  1. 録画ソフトのインストール
    要るもの：旧版 アマレコTV Live ( 直リン:amarectv310.zip、amv410.zip ) の2つ
    インストールの順は amv410.zip の amv4_setup_64bit.msi をインストール
    amarectv310.zip をCドライブの適当な所にフォルダを作って解凍したら
    live_setup300.exe をインストールし、AmaRecTV.exe を右クリックしスタートにピン留めする。

  2. 録画ソフトの設定
    アマレコの何も映ってない所を右クリックして設定、デスクトップキャプチャモード にする
    保存先は Cドライブ以外 が望ましい( 特にノートの場合 )、効果音のチェックを外しておく事。


    グラフ1 は ビデオキャプチャデバイスに AmaRec DesktopCapture を選択
    オーディオキャプチャデバイスに AmaRec Stereo Mixer の スピーカー を選び 48KHz にする。


    グラフ2、グラフ3、プリセットは特に設定せずに 録画設定 で フレームレート 999 にしたら
    ビデオ圧縮 AMV4、オーディオ圧縮 Uncompress でおｋ


    ホットキーのタブで ホットキーを使う のチェックを外して 適用 → OK
    設定がオワたら一度アマレコを閉じて再起動


  3. 取り込み
    アマレコのメインウィンドウで青い四角のアイコンを右クリックしてウィンドウキャプチャモードにする
    マウスカーソル録画、フレームレートは60にしておいておｋ


    キャプチャモードを変更したら青い四角いアイコンを普通にクリックし、青い枠をROのウィンドウに合わせて
    左上の白い□□が並んでるアイコンの右側をROウィンドウにドラッグ＆ドロップし、右下のSETをクリック
    アマレコ本体は下なり横なりにはみ出して置いとけばおｋ ※1366x768のノートだと置き所がなくてキツイ


  4. 録画 (次回起動以降はここから)
    赤い |＞ をクリックすればクッソでかい録画ファイルが保存される / 赤い□で録画停止
    ROが入ってるドライブと録画ドライブを物理的に別にしないとローディングでクッソ重くなるので注意
    1台のドライブのパーティションを区切って C:/D: 分けたｗｗ余裕ｗｗｗってのは天才過ぎるから不可

    ドライブの性能が低いと録画した動画ファイルをクリックして見てもカックカクになる場合があるが
    aviutlに読ませてエンコードしたらちゃんと記録されてる場合もある

    スペックが足りずに重かったりコマ落ちしてた場合は青い四角いアイコンを右クリックして
    フレームレートを30にしてみたり、ROウィンドウ自体を800x600にして録画すると改善する場合がある※
    （ ※限度がある ）

    EQの動画を録った時は全部のファイルで100GB位だった

  5. その他
    /bgm でBGM切っといた方が後で楽
    RO内でrecも同時にやっとくと撮り直しも楽だが、2回見るのめんどいから直で録っちゃってもおｋ
    直で録る場合、カーソルは自キャラやPCを避けて動かすと後で楽 ( PT欄からスキルを使用したり )

    基本的にPCで鳴る音、エラー音とかブラウザの音とかdiscordの通知とかの音も入るので
    閉じるなり取り込み中にしてどうぞ

    amv の透かしが入るのはレジストするか、utvideo を入れて録画設定で
    コーデックいろいろ→更新→UMRG でおｋ
